# Embed Image Sample

## Command Option

```bash
$ md2html example/embed-image.md -e
```

### Local file is embedded

```markdown
![logo/nocd5](img/go.png "nocd5")
```

![logo/nocd5](img/go.png "nocd5")

#### Write HTML tag immediately

```markdown
<img src="img/go.png" width="48">
<img src="img/go.png" width="32">
<img src="img/go.png" width="16">
```

<img src="img/go.png" width="48">
<img src="img/go.png" width="32">
<img src="img/go.png" width="16">
